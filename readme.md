# Talkr Chat App

Introducing Talkr - web chatting app, built using React.js, Node.js & Socket.io (used for bi-directional communication using TCP protocol).

## Features

- User can create separate chat room with unique room ID allow only selected user.

- Other user(s) can join the room with shared room ID.

- Host & other user will get notified whenever new user entered into room or someone left the room.
